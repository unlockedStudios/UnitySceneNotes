# Changelog

Notable changes to Scene Notes.

## [Unreleased]

- Added EditMode tests for category colors, section filtering, all-section visibility, selected note draw order,
  section deletion fallback, and hidden-note cases.
- Removed Odin Inspector assembly references from the runtime package so `Scene Note` appears in consuming projects
  without requiring Odin.

## [1.0.0] - 2026-06-08

- Added the initial Unity Package Manager package layout.
- Documented Odin Inspector as a prerequisite instead of bundling it with the package.
